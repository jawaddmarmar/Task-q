# 🚀 Task-Q – Smart Project & Task Management System

Task-Q is a modern web-based project and task management system designed to help teams organize projects, assign tasks, monitor progress, and collaborate efficiently through a simple and intuitive interface.

The system provides role-based access, project organization, task tracking, reporting, and team management, making it suitable for students, small businesses, and organizations.

---

## 📌 Features

### 🔐 Authentication & Security

* Secure user authentication
* Session management
* Role-based access control
* Protected dashboard pages

### 👥 Team Management

* Create and manage teams
* Add and organize team members
* Assign members to projects
* View team information

### 📁 Project Management

* Create new projects
* Edit and update project details
* Track project progress
* Manage project status
* Assign project members

### ✅ Task Management

* Create tasks
* Assign tasks to team members
* Set task priorities
* Define deadlines
* Update task status
* Track completion progress

### 📊 Dashboard & Reports

* Interactive dashboard
* Project statistics
* Task overview
* Progress monitoring
* Reports for projects and tasks

### 🎨 User Interface

* Clean and responsive design
* Easy navigation
* Modern dashboard
* User-friendly experience

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/jawaddmarmar/Task-q.git
```

2. Move the project into your XAMPP `htdocs` directory.

3. Start **Apache** and **MySQL** using XAMPP.

4. Import the database file into phpMyAdmin.

5. Update the database connection if necessary.

6. Open your browser and navigate to:

```text
http://localhost/Taskq/project_section/
```
